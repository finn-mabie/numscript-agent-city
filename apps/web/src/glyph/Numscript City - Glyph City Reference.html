GLYPH CITY · VISUAL SYSTEM v0.1

Numscript City.

A typographic micro-economy. Ten autonomous agents transact on a Formance ledger; every character, every coin, and every cage barrier is rendered as glyph on a monospace grid. No pixel art. No primitives. One continuous surface — the canvas, the chrome, the ticker, the intent-board — all rendered in the same ink.

02HERO

The barrier-slam.

Judy tried to debit $900 from a bank vault she doesn't own. The cage caught it before it reached the ledger. This is the screenshot.

TX 4,414 · REJECTED

Read. Left zone is the Bank — Dave dim at his post, vault readout calm. Right zone is marked unknown origin with red diagonals; Judy glows inside it. Between them the cage dialog has slammed down, slightly rotated, corner-cut like a rubber stamp. Every field inside the dialog is a monospace key-value: tx, by, debit, avail, short.

Why this moment. The entire thesis of the demo is "the cage protects every transaction." A committed transaction doesn't prove that. A rejected one does. The hero composition makes the enforcement the protagonist — the agents are supporting cast.

04LEXICON

Six buildings as ASCII sigils.

Each building is a 5-line block of box-drawing characters — pick-up-and-type it in Slack, paste it into a changelog, drop it into a docs header. The tint zone behind it carries the signature hue; the _CODE/ tag names it in the mono grammar of the rest of the surface.

05FLOW

Money in motion is a receipt.

When a transaction fires, a three-line posting-receipt pops at the receiver. Amount, counterparty, outcome — in that order. Committed receipts pin mint-green on their left edge, in-flight pins gold, rejected pins red. The pin IS the state.

06BARRIERS

Four ways the cage says no.

Every barrier is the same frame — ┌─ CAGE ─┐ top, error code top-right, key-value body, hairline rule — but each keeps a signature hue, a signature sigil, and its own last line of explanation. They coexist. None of them screams louder than the others.

08CHROME

No boxes. Hairlines only.

The canvas does not sit inside a panel inside a page. Top rail, canvas, intent-board, ticker — all on the same surface, separated by single 1 px hairlines. The result reads as one Bloomberg-adjacent terminal that happens to contain a village.

LAYOUT · WHOLE PAGE

Rules of the surface. One background color. One grid of dots. Hairlines are always #0a4048. Accent colors only enter as glyphs, pins, or signature zone tints — never as panel fills. No backdrop-blur, no shadows, no rounded corners — they would all re-introduce the "box inside box" feel we dissolved.

Modulation. Density varies by region — the canvas is loose (agents breathing, tint zones mostly empty), the intent-board is dense (ticker rows stacked tight). Contrast by information density, not by container style.

BEFORE → AFTER

  • pixel tiles → typed zones
  • phaser primitives → ASCII buildings
  • boxed HUD → top rail
  • side panels → hairlines
  • coin particles → posting receipts
  • flash rejections → stamped dialogs
  • bubbles → gold/silver bordered quotes
  • random walks → cell-tweened glides
09NEXT

What we build first.

This sheet is the reference. The next artifact is a playable scene — ten agents walking on the glyph grid, three live transactions per tick, Judy probing once every twenty ticks, all four cage barriers firing visibly within ninety seconds.

PHASER · ON-TOP

Keep. Phaser, the random-walk + home-biased motion system, the ticker event bus, the existing ledger stub. All the mechanics survive; only the renderers change.

Replace. Every 16 × 16 tile with a single glyph rendered via scene.add.text in Berkeley Mono at 24 px, snapped to the 24 px canvas grid, tweened between cells over 240 ms. Every primitive-shape building with a text block laid out in a 3 × 3 cell zone, tinted by its signature hue at 14 % alpha.

Add. One barrier-dialog component that reads a rejection event and renders a framed stamp at the failure site. One posting-receipt component that floats at the receiver for 1.8 s. One intent-board panel that lives flush-right, not in a window.

Don't add. Illustration. Portraits. Roofs. Shadows. Gradients. Sans-serif. The whole direction's discipline is only typography + color.
